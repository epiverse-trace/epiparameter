# Format short citation from `<bibentry>` object

Output is equivalent to the `\citet{}` function in the natbib LaTeX
package.

## Usage

``` r
.citet(x)
```

## Arguments

- x:

  A `<bibentry>` object, see
  [`bibentry()`](https://rdrr.io/r/utils/bibentry.html).

## Value

A `character` string with the short citation.
