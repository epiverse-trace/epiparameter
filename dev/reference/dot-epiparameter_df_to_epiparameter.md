# Convert `<data.frame>` from [`as.data.frame.epiparameter()`](https://epiverse-trace.github.io/epiparameter/dev/reference/as.data.frame.epiparameter.md) to `<epiparameter>`

Convert `<data.frame>` from
[`as.data.frame.epiparameter()`](https://epiverse-trace.github.io/epiparameter/dev/reference/as.data.frame.epiparameter.md)
to `<epiparameter>`

## Usage

``` r
.epiparameter_df_to_epiparameter(x, ...)
```

## Arguments

- x:

  A `<data.frame>`.

- ...:

  [dots](https://rdrr.io/r/base/dots.html) Extra arguments to pass to
  [`epiparameter()`](https://epiverse-trace.github.io/epiparameter/dev/reference/epiparameter.md).

## Value

An `<epiparameter>` object.
